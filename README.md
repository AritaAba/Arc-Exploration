# 环形探索 —— 产品介绍文档

> 一站式旅游 / 出差信息资讯查询平台，让每一次出行都成为值得珍藏的记忆。

---

## 一、产品定位

**环形探索** 是一款以交互式三维地球为核心的旅行规划与信息资讯平台。用户可以在沉浸式的星空背景下自由探索世界各国，一键获取目的地的地理、人文、交通、路线等全方位信息，同时通过内置的照片收藏与旅行随笔功能，将规划过程本身变成一段有温度的旅行纪念。

**核心价值主张**

| 维度 | 描述 |
|------|------|
| 信息聚合 | 整合地理、气候、饮食、节日、景点、交通、路线等多类数据，一页直达 |
| 沉浸体验 | 可交互三维地球 + 星空背景 + 动态光效，让查询本身成为享受 |
| 旅行纪念 | 内置照片胶卷与旅行随笔，陪伴行前规划到行后回忆的完整旅程 |
| 轻量无后端 | 单一 HTML 文件，本地存储数据，无需注册，打开即用 |

## 二、功能模块总览

### 2.1 首页 —— 交互式三维地球

用户打开页面后，首先看到的是一个缓缓自转的三维地球，悬浮在深邃的星空之中。

**视觉效果**
- **星空背景**：数千颗粒子组成的星场，配合自定义着色器实现逐星闪烁、漂移与呼吸动效。
- **大气层光晕**：地球外围三层半透明光晕随时间脉动，模拟真实的大气散射。
- **国家边界**：基于 Natural Earth 110m 矢量数据实时渲染的国界线，覆盖全球所有独立国家。
- **实时时钟**：右上角 HUD 显示当前本地时间，逐帧刷新。

**交互方式**
- **点击进入**：点击地球切换至全屏探索模式。
- **拖拽旋转**：按住拖动可自由旋转地球，松手后保留惯性继续滑动。
- **滚轮缩放**：平滑缩放地球，配合阻尼过渡。
- **双击返回**：双击地球退回首页英雄视图，恢复自动旋转。
- **搜索定位**：输入国家或城市名称，地球自动旋转对准目标位置并弹出信息面板。

---

### 2.2 国家信息面板

点击地球上的任意国家（或通过搜索命中），右侧弹出该国家的**快速概览面板**。

**面板内容**

| 区块 | 信息项 |
|------|--------|
| 头部 | 国旗图标、中文国名、首都名称 |
| 地理信息 | 经纬度范围（度分秒中文格式）、国土面积 |
| 人口与社会 | 人口数量、官方语言（中文标注） |
| 经济与货币 | 货币名称、符号（中文标注） |
| 交通与通讯 | 国际区号、时区列表、与本地时区的时差、靠右/靠左行驶 |

面板支持**智能定位**——自动检测视口边界，确保面板不会超出屏幕范围。

点击面板中的**"更多"**按钮，进入该国家的专属详情页面。

---

### 2.3 国家详情页 —— 旅行规划中心

详情页采用全屏覆盖层设计，左侧为地图与路线模块，右侧为信息选项卡与收藏区。

---

#### 2.3.1 地图模块

- 渲染该国家的**精细矢量轮廓**（基于 Natural Earth 50m 数据），以及周边邻国的简化边界作为参照。
- 在地图上标注**首都及主要城市**（最多 10 座），坐标数据通过 Open-Meteo 地理编码实时获取。
- 支持**平移与缩放**操作，适合近距离查看地理细节。

---

#### 2.3.2 路线模块 —— "出发 · 路径"

帮助用户规划从当前位置（或指定城市）前往目的地的行程方案。

**出发地设置**
- **当前位置**：调用浏览器定位 API，自动获取所在城市坐标。
- **搜索城市**：手动输入出发城市，通过 Open-Meteo 地理编码解析坐标。
- 出发地信息持久化保存在本地，下次打开自动恢复。

**交通方式选择**

| 方式 | 距离/时长计算方法 |
|------|------------------|
| 飞机 ✈ | Haversine 球面距离 + 巡航速度估算（约 820 km/h），含起降时间 |
| 高铁·火车 🚄 | Haversine 距离 + 平均速度估算 |
| 汽车 🚗 | 调用 OSRM 真实路网数据，返回驾车距离与预计时长 |
| 轮船 🚢 | 球面距离 + 航速估算 |
| 骑行 🚲 | 球面距离 + 骑行速度估算 |
| 步行 🚶 | 球面距离 + 步行速度估算 |

**行程摘要**
- 显示精确距离（公里）、预计总时长。
- **飞机模式**额外提供：综合枢纽机场推荐、模拟航班号、本地起降时刻表（基于经度换算时区）。
- 世界地图预览：等距矩形投影显示出发地至目的地的**大圆弧路线**。

**出发时间设置**
- 支持设置计划出发日期与时刻，动态更新行程摘要中的时间节点。

---

#### 2.3.3 三大信息选项卡

详情页右侧设有三个切换选项卡，覆盖行程规划所需的核心信息维度。

---

##### 行程（默认）

整合**路线模块**的所有交互内容：出发地设置、交通方式选择、出发时间设定、行程摘要，以及世界地图路线预览。

---

##### 风俗

展示目的地国家的人文风俗信息，帮助用户提前了解当地文化。

| 板块 | 内容示例 |
|------|----------|
| 气候·各地区 | 不同地区的典型天气特征、气温范围、最佳旅行季节 |
| 饮食·知名食物 | 代表性传统菜肴、街头小吃、饮食文化特点 |
| 节日·主要庆典 | 重要传统节日名称、举办时间、文化含义与庆祝方式 |

内容面板支持独立滚动，适合信息较多的国家。

---

##### 景点

列出目的地国家的知名旅游景点，每个景点包含：
- 景点名称
- 所在位置（省份 / 城市）
- 景点类型与简介

内容面板同样支持独立滚动。

---

### 2.4 照片胶卷（Film Stash）

照片胶卷模块始终固定显示在右侧下方区域，上传记录后将点亮该区域版图。

- **上传文件**：支持图片（jpg/png/gif/webp 等）、视频、PDF、Word 文档、文本文件、Markdown、压缩包等格式。
- **本地存储**：文件以 Blob 形式存储在浏览器 IndexedDB 中，单文件上限 120MB，无需上传至服务器。
- **胶卷预览**：横向滚动的缩略图条，点击任意缩略图全屏预览（图片大图、视频播放控件、文档下载链接）。
- **删除管理**：每个缩略图可单独删除，即时从本地数据库移除。

---

### 2.5 旅行随笔（Memo）

随笔模块与胶卷并列固定显示，作为行前思考与行后记录的私人空间，上传记录后将点亮该区域版图。

- **新建 / 删除**：随时创建多篇随笔，或删除不需要的条目。
- **标题 + 正文**：每篇随笔包含标题栏与自由文本编辑区。
- **自动保存**：内容变更后约 380ms 自动保存至浏览器 localStorage，关闭页面不丢失。
- **目录导航**：左侧随笔列表支持独立滚动，点击条目切换编辑，内容区随滚动完整展示。

## 三、技术亮点

| 技术 | 说明 |
|------|------|
| **Three.js 0.160** | 3D 地球渲染、国界线几何体、大气层光晕、星空粒子着色器 |
| **REST Countries API** | 国家基础数据（国旗、语言、货币、时区等） |
| **Open-Meteo Geocoding** | 全球城市地理编码，支持中英双语查询 |
| **OSRM** | 真实路网驾车路线规划 |
| **Natural Earth GeoJSON** | 110m / 50m 矢量边界数据 |
| **IndexedDB** | 本地大文件（媒体素材）持久化 |
| **localStorage** | 路线设置与随笔数据持久化 |
| **CSS 动画** | 星云浮动、极光移位、标题光晕、地图边界动效 |
| **单文件架构** | 全部代码内联于单一 HTML，零依赖安装，本地直接打开 |

## 四、适用场景

- **出境旅游前期规划**：查询目的地国家概况、了解风俗禁忌、筛选知名景点。
- **商务出差准备**：快速了解目的地时区、货币、区号等商务信息，规划出行路线。
- **地理与文化学习**：通过交互式地球探索世界各国地理、语言、节日等文化知识。
- **旅行纪念留存**：上传旅途照片与视频，记录旅行随笔，打造专属的旅行数字纪念册。

## 五、快速上手

1. 用浏览器直接打开 `earth-explore.html`（推荐 Chrome / Edge / Firefox 最新版）。
2. 在首页搜索框输入目的地国家或城市名称，或直接点击地球进入探索模式后点选目标国家。
3. 在弹出的国家面板中点击**"更多"**，进入详情页。
4. 切换**行程 / 风俗 / 景点**选项卡获取所需信息，设置出发地与交通方式规划路线。
5. 在右侧**胶卷区域**上传旅行相关文件，在**随笔区域**记录行程想法。

> **注意**：定位功能（当前位置）、驾车路线（OSRM）等部分功能在 `file://` 协议下可能受 CORS 限制，建议通过本地静态服务器（如 VS Code Live Server）访问以获得完整体验。
> 由于数据库原因，当前测试界面的信息可能不全或不足。

## 六、数据与隐私

- 所有个人数据（上传文件、随笔内容、路线设置）**仅存储在用户本机浏览器**，不会上传至任何服务器。
- 对外网络请求仅用于获取公开的国家信息、地理编码、路线规划及字体资源，不含任何用户标识信息。
