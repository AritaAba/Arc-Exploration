# Arc-Exploration

> 为旅行者与出差人士打造的一站式旅行规划信息集成平台。
> 欢迎收看demo视频：https://youtu.be/tYysIvBMSns?si=CPo46GjsYiSqOxgE

---

## 产品简介

**Arc-Exploration** 是一个以交互式 3D 地球为入口的旅行规划工具，面向有出境旅游或商务出差需求的用户。它将目的地国家的地理概况、交通路线、风俗文化、热门景点等分散信息整合到同一个页面中，帮助用户在出发前高效完成信息收集与行程规划。内置的 **AI 路线助手**可以自由问答，解答签证、行程、文化、交通等各类旅行问题。同时，照片上传与旅行随笔功能允许用户为每个到访国家留下专属记录，让规划过程本身也成为旅行记忆的一部分。

## 核心功能

### 1. 交互式 3D 地球（首页）

首页是一个缓缓自转的三维地球，悬浮于粒子星空背景中。

- **探索模式**：单击地球进入自由旋转视图，支持拖拽旋转（带惯性）与滚轮缩放。
- **搜索定位**：输入国家或城市名称（支持中英文），地球自动对准目标并弹出信息概览。
- **国家点击**：在探索模式中点击任意国家，弹出信息面板。
- **双击返回**：双击地球回到初始视图，恢复自动旋转。
- **到访标记**：当用户在某国家详情页上传过照片或写过随笔后，该国家在 3D 地球上会被白色高亮填充，直观展示已探索的足迹。

### 2. 国家信息面板

点击或搜索到某个国家后，弹出快速概览面板，包含：

| 信息类别 | 内容 |
|---------|------|
| 基础信息 | 国旗、中文国名、首都 |
| 地理 | 经纬度范围、国土面积 |
| 人口与语言 | 人口数量、官方语言 |
| 经济 | 货币名称与符号 |
| 交通与通讯 | 国际区号、时区列表、时差、行驶方向（靠左/靠右） |

点击面板中的 **"更多"** 按钮进入该国家的完整详情页。

### 3. 国家详情页

详情页为全屏布局，左侧为地图与路线，右侧为信息选项卡、AI 助手与个人记录区。

#### 3.1 地图模块

- 基于 Natural Earth 50m 矢量数据渲染国家精细轮廓及邻国边界。
- 标注首都与主要城市（坐标通过 Open-Meteo 地理编码实时获取）。
- 支持平移与缩放。

#### 3.2 行程规划（"行程"选项卡）

帮助用户规划从出发地到目的地的交通方案。

**出发地设置**：支持浏览器定位自动获取当前城市，或手动搜索输入。出发地信息本地持久化保存。

**六种交通方式**：

| 方式 | 说明 |
|------|------|
| 飞机 ✈ | 球面距离 + 巡航速度估算，含枢纽机场推荐、模拟航班号、本地起降时刻 |
| 高铁/火车 🚄 | 距离与平均速度估算 |
| 汽车 🚗 | 调用 OSRM 真实路网数据，返回实际驾车距离与时长 |
| 轮船 🚢 | 球面距离 + 航速估算 |
| 骑行 🚲 | 球面距离 + 骑行速度估算 |
| 步行 🚶 | 球面距离 + 步行速度估算 |

**行程摘要**：显示距离（公里）、预计时长，世界地图上以大圆弧线展示路线。支持设置出发日期与时刻。

#### 3.3 风俗文化（"风俗"选项卡）

展示目的地国家的人文信息，帮助用户提前了解当地环境：

- **气候**：各地区典型天气与气温范围
- **饮食**：代表性食物与饮食文化
- **节日**：主要传统节日的时间、含义与庆祝方式

#### 3.4 热门景点（"景点"选项卡）

列出目的地国家的知名景点，每项包含景点名称、所在位置（省份/城市）及简介。可勾选多个景点后一键生成串联路线，由 AI 助手输出完整规划。

#### 3.5 AI 路线助手

详情页右侧常驻 **AI 路线助手**，由 DeepSeek 大语言模型驱动，支持自由对话与多轮追问。

**能力范围**：

- 签证办理流程与所需材料
- 最佳出行季节与气候建议
- 从出发地到目的地的交通方案
- 机场到市区的接驳方式
- 定制 N 天行程建议
- 景点串联路线规划（结合已勾选景点）
- 当地美食、文化禁忌、安全注意事项
- 货币兑换、消费水平参考
- 以及任何其他旅行相关问题

**特性**：

- 流式输出，逐字显示回复，体验流畅
- 多轮对话，记住上下文，可连续追问
- 自动携带当前目的地的景点、气候、文化等背景信息，回答更精准
- 切换目的地国家时自动重置对话上下文

#### 3.6 照片与文件上传

固定显示在详情页右下方，不随选项卡切换消失。

- 支持图片、视频、PDF、文档、压缩包等多种格式。
- 文件以 Blob 存储在浏览器 IndexedDB 中（单文件上限 120MB），无需服务器。
- 横向胶卷缩略图，可点击预览或删除。
- **各国数据独立**：在不同国家详情页上传的文件互不影响。

#### 3.7 旅行随笔

与照片模块并列固定显示。

- 支持创建多篇随笔，每篇含标题与正文。
- 内容变更后自动保存至 localStorage。
- 左侧目录列表可切换不同随笔。
- **各国数据独立**：每个国家的随笔互不影响。

#### 3.8 到访点亮提示

照片上传区上方显示提示："当记录照片或随笔后，地球上的这个区域将被你点亮！"——上传照片或写下随笔后返回首页，该国家区域将在 3D 地球上以白色填充高亮。

## 技术栈

| 技术 | 用途 |
|------|------|
| Three.js | 3D 地球渲染、国界线、大气光晕、星空粒子着色器、到访国家填充 |
| DeepSeek API | AI 路线助手，流式对话，支持多轮上下文 |
| REST Countries API | 国家基础数据（国旗、语言、货币、时区等） |
| Open-Meteo Geocoding | 全球城市地理编码（中英双语） |
| OSRM | 真实路网驾车路线规划 |
| Natural Earth GeoJSON | 110m / 50m 矢量国界数据 |
| IndexedDB | 本地媒体文件持久化（按国家隔离） |
| localStorage | 随笔与路线设置持久化（按国家隔离） |
| GLSL 自定义着色器 | 到访国家白色填充渲染、星空粒子动效 |
| 单文件架构 | 全部代码内联于 HTML，零依赖安装，本地打开即用 |

## 适用场景

- **出境旅游规划**：查询目的地概况、AI 问答签证与行程、了解当地风俗与气候、筛选景点、规划交通路线。
- **商务出差准备**：快速获取时区、货币、区号、行驶方向等实用信息，向 AI 咨询出行方案。
- **旅行记录与回忆**：为每个到访国家上传旅途照片、记录随笔，3D 地球上直观展示足迹版图。

## 快速开始

1. 使用浏览器打开 `index.html`（推荐 Chrome / Edge / Firefox 最新版）。
2. 在搜索框输入目的地名称（支持中英文），或点击地球进入探索模式后点选国家。
3. 在弹出面板中点击 **"更多"** 进入详情页。
4. 通过 **行程 / 风俗 / 景点** 选项卡获取所需信息，或直接向右侧 **AI 助手**提问。
5. 在右侧上传旅行照片、撰写随笔，返回首页查看地球上被点亮的国家。

> **提示**：浏览器定位和 OSRM 驾车路线等功能在 `file://` 协议下可能受跨域限制，建议通过本地静态服务器（如 VS Code Live Server）或部署至 GitHub Pages 后访问以获得完整体验。

## 数据与隐私

- 所有个人数据（上传文件、随笔、路线设置）**仅存储在用户本机浏览器**，不会上传至任何服务器。
- AI 对话内容通过加密 HTTPS 连接发送至 DeepSeek API，不含任何用户身份标识信息。
- 对外网络请求仅用于获取公开的国家信息、地理编码、路线规划、AI 回复及字体资源。
